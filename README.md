# EnoddGameEngine
Простой игровой 2D/3D движок


| **название папки**         | **для чего папка**                                |
| -------------------------- | ------------------------------------------------- |
| Engine                     | исхоные файлы движка                              |
| Engine/EngineEntity        | общий класс объектов подписанный на обновления    |
| Engine/EngineEntity/Window | общий класс окна                                  |
| Engine/GameEntity          | общий класс игровых сущностей                     |
| Engine/GameLoop            | класс игрового цикла                              |
| Engine/Render              | класс рендера                                     |
| Engine/Inputs              | класс ввода из переферии(ввод из клавиатыры/мыши) |
|                            |                                                   |
|                            |                                                   |
